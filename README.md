

### Downloading the library:

To download the library, run the following command:

```bash
curl -o httpserverhtml.py https://raw.githubusercontent.com/saarors/http-server-for-html/refs/heads/main/httpserverhtml.py
```

### Step 1: Create an `index.html` file

Create an `index.html` file in your directory with basic HTML content.

### Step 2: Run the following command in the terminal:

Afterward, run this command in the terminal:

```bash
python3 httpserverhtml.py
```

### Step Three:
Open your browser at 127.0.0.1:8080









 
Note: Only works on Linux
